Windows To Go最佳拍档：Sandisk CZ80 64G——安装VHD+WTG傻瓜教程

By Scott Yang

发表于 2014-12-29

文章目录

本文节选编辑自《13秒启动！Windows To Go最佳拍档：Sandisk CZ80 64G开箱+速度测试+安装VHD+WTG傻瓜教程（带附件）》，这里只挑干货整理，为过两天可能会买的cz80做准备。

Windows to go（下文简称WTG）作为windows8时代的全新功能，可以使完整版本的win8从USB驱动器启动及运行，自问世之初就受到了巨大的关注。由于WTG时系统运行于USB驱动器（U盘&移动硬盘）上，磁盘性能完全依附于所使用的设备，容量过小及速度过慢的U盘/移硬都会严重影响该系统的体验，因此微软还特地授予了部分U盘WTG认证（其实就是容量＞32G+速度较快的USB3.0 U盘而已）。

WTG系统有什么作用？

  1. 作为安装系统时使用的启动盘（在系统里放一个NT6.1安装器+diskgenuis即可）；
  2. 使用Acronis True Image在不同硬盘间迁移系统时可以使用WTG系统过度，不需制作专用的USB启动U盘，且更快速；
  3. 在系统里安装较为冷门的办公软件（OCR软件、PDF转格式软件etc）后可以在不同的电脑上使用，不需要再次安装该软件（其实硬件跑分狂也可以用来装一堆3DMARK进行测试……）
  4. 需要演示文稿时可以载入自带的系统，不用担心演示用的机子所使用的软件版本不同所引起的的排版错误
    相比WTG刚推出的时候，现在VHD下安装WTG过程已经十分简单，安装完纯净系统之后也还有近30+G可用于存放其他文件，现将所需文件及傻瓜教程附上。

既然是傻瓜教程，那就先复制一段关于VHD的百科。

VHD是Microsoft Virtual Hard Disk format（微软虚拟磁盘文件）的简称，具体地说，即是创见出虚拟成硬盘分区的文件，系统可从该虚拟磁盘中启动。而VHD不使用时也可以被压缩成一个文件存放在宿主机器的文件系统上。

使用VHD安装WTG的好处是不需在U盘划分系统专用分区，且易于管理，不需要时可随时删除，同时可以在原有系统下挂载VHD查看其中的文件。

WTG刚推出的时候的VHD安装需要自己手动创建VHD，在非WIN8企业版的系统下还需手动安装系统后bootice引导启动，安装系统之后还要kms破解……但现在已经有了傻瓜程序，WIN7/WIN8下均可运行。

然后点击绿框处选中Sandisk Extreme USB Device（这个千万别选错了）

最后在蓝框处选中VHD模式。
