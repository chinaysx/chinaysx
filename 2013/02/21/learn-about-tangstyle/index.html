文章目录

无意中发现了这个非常对我口味的主题——TangStyle，这篇文章就简单记录一下对这个主题的学习和修改吧
之前没太研究过php，现在也只能干点鸡毛蒜皮缝缝补补的小活了，不过也总算是干了
1、侧边栏的修改
侧边栏唐大的默认设置做的很出色了，能做的改动也只是改一下栏目名，调整一下顺序，对了，还放了一个微博的小挂件
相关的文件 sidebar.php sidebar-category.php sidebar-single.php
未完成的项目
计划中 流行 模块下的文章数要改成5篇，这个还没做……
删除线的颜色还不知道去哪改
2、文章顺序
模版的默认排序是按照发布时间来的，简单的修改了一下，改成按最后编辑时间排序
3、解决图片环绕问题
在wordpress编辑器中能够自由控制图片的居左居右居中显示，而且是文字围绕着图片，然而到了前台就无法显示
解决方法
在style.css中加入如下代码

/*Alignment*/
.alignleft {
display:inline;
float:left;
margin-right:1.625em;
}
.alignright {
display:inline;
float:right;
margin-left:1.625em;
}
.aligncenter {
clear:both;
display:block;
margin-left:auto;
margin-right:auto;
}
