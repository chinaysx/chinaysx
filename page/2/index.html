我的搞机史

By Scott Yang

发表于 2015-04-29

想想自己的刷机史，从06年到现在，马上就快十年了。十年，弹指一挥间。
06年，从老爸的手中接过了我的第一台手机，Moto E398，那台机器到我手上的时候，已经被我爸刷过了，Java系统，能跑一些Java小程序，那个时候就有很多人在做E398刷机包，加入了各种功能，刷机工具也很简陋，而且全都是线刷。值得一提的是，那时候就已经有了内置软件和反内置软件了—
过了两年，E398在我的摧残之下，终于宣告退役，所以从我老爸手里接过了我的第二台手机——Moto L7，也是当时的一个经典啊，还有30W的摄像头，直板硬朗的机身，当然还有数不清的刷机包，每个包都在追求各自的特点，结果每个包都大同小异。
对了，那个时候有个土豪同学用着行货的Moto A1200，由于玩着不爽被我抢过来给刷了—然后我们度过了一段非常开心的日子
再后来，上了大学之后，终于有了第一台属于自己的手机，自己挑选的一手机器——Moto Milestone。一代的里程碑，想来不知道从什么时候开始我就已经潜移默化的成为Moto的粉丝了，我是身边的人里面第一个用安卓智能机的，在那个满世界都是诺基亚的时代，有一种–特立独行–的感觉。我差不多问遍了淘宝的每一家卖水货里程碑的店家，最后花了2000多才买下了那台机器。里程碑陪了我三年，到最后因为屏幕失灵而退役。
用里程碑的时候，每天沉浸于机锋-安智网的milestone板块和软件应用板块，还有一个必去的论坛——itfunz，看着很面熟么，是的，后来他改名了，叫做mfunz。那时候每天最期待的就是等着龙猪发包，每天在那里水啊水，直到龙猪放弃了milestone。很多年以后，当我因为一加的缘故再次进入mfunz，用当年的账号居然登陆进去的时候，感慨良多。
miui也是那个时候开始用的，那个时候小米1还没有发布，从itfunz里看到了网友喊人去miui投票求适配的帖子——和今天又是何等的相似。miui刚刚起步，但是一些功能的确是深得我心。那个时候我觉得miui真的是国内最好用的ROM了。小米1发布，30W台预售，我抢了一个号，送给了舍友。从那时起我好像也没再怎么用过miui，也许是因为milestone已经跑不动越来越臃肿的miui，也许是miui的那种风格的确不适合我。
插一句，其实搞机也并不仅仅局限于手机，毕竟我还刷过电子词典，并且在相当长的一段时间内沉迷于各种路由器，早在各种智能路由器出来之前，我的路由器就比他们的还要智能了。
告别了我的第一台自己挑选的手机，有继续接过了老爸的二手机——联想P770，他换了HTC的M7，说起来，p770是我为他选的，两三年前就有3500mah的大电池，配上mtk的处理器，不说性能，论续航，应该是数一数二的了。那个时候老爸早已经退出刷机圈了，所以手机到我手上的时候，依然是原生的联系系统，乐UI，好像是这个名字吧。当然，因为看不惯那么多的内置软件，到我手上的第一天，系统就被我刷掉了，印象中可能是换成了乐蛙。在网上找来找去，大概是找到了唯一还活跃的P770的论坛–移动叔叔。在那里，一位大神的包简直是每发必刷——suda。很高兴在一加这里依然能看到他。
从放弃了miui开始，我意识到自己还是属于那种更倾向于原生系统的那种人，追求着那种简单的生活方式。所以即使是手机上没有aosp的包，即使是刷的乐蛙，也要装上nova，至少看起来也像那么回事，简单，直接，实用才是正道。
因为学的是计算机专业，也许是由于专业的关系，一直比较科技圈的事，所以从一加品牌发布的时候就关注了下这个牌子，到手机发布会上说道一加将于CM合作的时候，那时候忽然觉得这个手机好符合我对手机的要求啊，只是遗憾P770太过坚挺，也没有什么换机的理由。
去年九月，终于，P770给了我一个机会，让我有借口入了倾心已久的一加。
机器到手的第一天，系统变成了CM11S，Color 1.2大概存活了不到4个小时。突然觉得好圆满。再后来，等到安卓5.0出来，感受着安卓的变化和MD设计的美学。氧OS的发布，更是让手中的一加有一种成了谷歌干儿子的感觉。CM12S更是锦上添花了。
一加天气是一款强力筛选一加用户的应用，我很喜欢这句话，也对氢有了更多的信心。因为志同道合，所以我情愿做一加的自干五。
希望我们不会让彼此失望。

Windows To Go最佳拍档：Sandisk CZ80 64G——安装VHD+WTG傻瓜教程

By Scott Yang

发表于 2014-12-29

本文节选编辑自《13秒启动！Windows To Go最佳拍档：Sandisk CZ80 64G开箱+速度测试+安装VHD+WTG傻瓜教程（带附件）》，这里只挑干货整理，为过两天可能会买的cz80做准备。

Windows to go（下文简称WTG）作为windows8时代的全新功能，可以使完整版本的win8从USB驱动器启动及运行，自问世之初就受到了巨大的关注。由于WTG时系统运行于USB驱动器（U盘&移动硬盘）上，磁盘性能完全依附于所使用的设备，容量过小及速度过慢的U盘/移硬都会严重影响该系统的体验，因此微软还特地授予了部分U盘WTG认证（其实就是容量＞32G+速度较快的USB3.0 U盘而已）。

WTG系统有什么作用？

  1. 作为安装系统时使用的启动盘（在系统里放一个NT6.1安装器+diskgenuis即可）；
  2. 使用Acronis True Image在不同硬盘间迁移系统时可以使用WTG系统过度，不需制作专用的USB启动U盘，且更快速；
  3. 在系统里安装较为冷门的办公软件（OCR软件、PDF转格式软件etc）后可以在不同的电脑上使用，不需要再次安装该软件（其实硬件跑分狂也可以用来装一堆3DMARK进行测试……）
  4. 需要演示文稿时可以载入自带的系统，不用担心演示用的机子所使用的软件版本不同所引起的的排版错误
    相比WTG刚推出的时候，现在VHD下安装WTG过程已经十分简单，安装完纯净系统之后也还有近30+G可用于存放其他文件，现将所需文件及傻瓜教程附上。

既然是傻瓜教程，那就先复制一段关于VHD的百科。

VHD是Microsoft Virtual Hard Disk format（微软虚拟磁盘文件）的简称，具体地说，即是创见出虚拟成硬盘分区的文件，系统可从该虚拟磁盘中启动。而VHD不使用时也可以被压缩成一个文件存放在宿主机器的文件系统上。

使用VHD安装WTG的好处是不需在U盘划分系统专用分区，且易于管理，不需要时可随时删除，同时可以在原有系统下挂载VHD查看其中的文件。

WTG刚推出的时候的VHD安装需要自己手动创建VHD，在非WIN8企业版的系统下还需手动安装系统后bootice引导启动，安装系统之后还要kms破解……但现在已经有了傻瓜程序，WIN7/WIN8下均可运行。
